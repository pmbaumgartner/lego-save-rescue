Browser-based PC save repair

LEGO Star Wars: The Skywalker Saga

Save Rescue

Upload a zipped save folder and download repair candidates for saves that fail to resume from a broken location state. Selected files are processed in your browser.

1. Zip the save folder

Before you start: close the game, back up SAVEDGAMES, and test on a copy.

Turn off Steam Cloud: Steam Library > right-click game > Properties > General > turn off Keep game saves in the Steam Cloud.

Find the numbered folder that contains PROFILEDATA and your SLOT# folders.

Zip that numbered folder. Parent folders are okay.

Common locations:

Windows / Steam
%APPDATA%\Warner Bros. Interactive Entertainment\LEGO Star Wars - The Skywalker Saga\SAVEDGAMES\STEAM\<your-number>\
Steam Deck or Linux / Proton
~/.local/share/Steam/steamapps/compatdata/920210/pfx/drive_c/users/steamuser/AppData/Roaming/Warner Bros. Interactive Entertainment/LEGO Star Wars - The Skywalker Saga/SAVEDGAMES/STEAM/<your-number>/

On some Linux installs the Steam folder starts at ~/.steam/steam/steamapps/compatdata/920210. For SD cards or alternate Steam libraries, start from that library's steamapps/compatdata/920210 folder.

2. Upload the zip

Choose the zip you made from that folder.

The original upload is copied into the repair package unchanged.

3. Generate candidates

Choose the broken slot. The output starts with the smallest location-only repair, then includes broader fallbacks.

Status

Choose a save folder zip, then click Analyze.

Test the repair package

The downloaded zip contains repair candidates under candidates/. Extract it, then test one candidate folder at a time.

  1. Start with 01-location-only-safe-landing.
  2. Copy that candidate's PROFILEDATA and SLOT# folders into your backed-up save-account folder.
  3. Launch the game and try the slot.
  4. If it fails, close the game and copy the next candidate over the same test save.

The first candidate patches only GAMEFLOW.BLOB and GAMEPROGRESS.BLOB. The second also updates SAVEGAMEMETA.BLOB. The full candidate resets PARTYSYSTEM.BLOB and PARTYFREEPLAYLIST.BLOB only as a last resort.

Important limits

This is not a universal save decoder. It targets PC or Proton saves where progress data still exists, but the current resume/location/session state is broken. It does not repair console saves or damaged collectibles and achievement files.

Selected zips are processed in your browser and are not sent to a server.
