Browser-based PC save repair

LEGO Star Wars: The Skywalker Saga

Save Rescue

Upload a zipped save folder to build a repair package for PC saves that fail to resume from a broken location state. Files are processed in your browser.

1. Find and zip the save folder

Before you start: close the game, back up SAVEDGAMES, and test on a copy.

Turn off Steam Cloud: Steam Library > right-click game > Properties > General > turn off Keep game saves in the Steam Cloud.

Start from the matching path for your system:

Windows / Steam
%APPDATA%\Warner Bros. Interactive Entertainment\LEGO Star Wars - The Skywalker Saga\SAVEDGAMES\STEAM\<numbered-folder>\
Steam Deck or Linux / Proton
~/.local/share/Steam/steamapps/compatdata/920210/pfx/drive_c/users/steamuser/AppData/Roaming/Warner Bros. Interactive Entertainment/LEGO Star Wars - The Skywalker Saga/SAVEDGAMES/STEAM/<numbered-folder>/

In the paths above, <numbered-folder> is the folder to zip. Open it and confirm it contains PROFILEDATA and folders like SLOT1 or SLOT2.

Zip that numbered folder. A zip that also includes parent folders is okay.

On some Linux installs the Steam folder starts at ~/.steam/steam/steamapps/compatdata/920210. For SD cards or alternate Steam libraries, start from that library's steamapps/compatdata/920210 folder.

2. Upload the zip

Choose the zip file you created in step 1.

Your original zip is included unchanged in the repair package.

3. Generate repair package

Choose the save slot that fails to load. The package starts with a focused location repair and includes broader fallback options.

Status

Choose a save folder zip, then click Analyze.

Test the repair package

The downloaded zip contains repair options under candidates/. Extract it, then test one option at a time.

  1. Start with 01-location-only-safe-landing.
  2. Copy that option's PROFILEDATA and SLOT# folders into the numbered save folder you backed up.
  3. Launch the game and try the slot.
  4. If it fails, close the game and copy the next repair option over the same test save.

The first option patches only GAMEFLOW.BLOB and GAMEPROGRESS.BLOB. The second also updates SAVEGAMEMETA.BLOB. The full option resets PARTYSYSTEM.BLOB and PARTYFREEPLAYLIST.BLOB only as a last resort.

Important limits

This tool is scoped to PC and Proton saves where progress data still exists, but the current resume, location, or session state is broken. It does not repair console saves, damaged collectibles, or achievement files.

Save files are processed locally in your browser.
